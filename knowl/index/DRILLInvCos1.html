Problem 6.81.
(a)
Use the fact that \(y=\arccos(x)\) if and only if \(\cos(y)=x\) and proceed as we did in Drill 6.80 to show that
\begin{equation} \dx\left(\arccos(x)\right)= \frac{\mp 1}{\sqrt{1-x^2}}\dx{x}.\tag{6.16} \end{equation}
Notice how this formula differs from the one you found in Drill 6.80.
(b)
Derive the same result from the identity: \(\arccos(x)= \frac{\pi}{2}-\arcsin(x)\text{.}\)
