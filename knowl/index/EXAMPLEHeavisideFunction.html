Example 16.1.0.10. The Heaviside Function.

Problem 16.1.0.5 shows that the derivative of the Absolute Value function is:
\begin{equation*} H(x)= \begin{cases} 1\amp x\gt0\\ -1\amp x\lt0 \end{cases}. \end{equation*}
Notice that it is not defined at \(x=0\text{.}\) We call \(H(x)\) the Heaviside function in honor of Oliver Heaviside
