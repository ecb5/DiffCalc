Problem 6.95.
(a)
Show that each of the following statements is true.
  1. \(\displaystyle \dx\left(\inverse\sin\left(\frac{x}{\sqrt{x^2+1}}\right)\right)=\frac{1}{x^2+1}\dx{x}\)
  2. \(\displaystyle \dx\left(\inverse\sin\left(\frac{x}{\sqrt{x^2+4}}\right)\right)=\frac{2}{x^2+4}\dx{x}\)
  3. \(\displaystyle \dx\left(\inverse\sin\left(\frac{x}{\sqrt{x^2+9}}\right)\right)=\frac{3}{x^2+9}\dx{x}\)
  4. \(\displaystyle \dx\left(\inverse\sin\left(\frac{x}{\sqrt{x^2+5}}\right)\right)=\frac{\sqrt{5}}{x^2+5}\dx{x}\)
(b)
Now compute \(\dx\left(\inverse\sin\left(\frac{x}{\sqrt{x^2+a^2}}\right)\right),\) where \(a\gt0.\) Does this problem change if \(a\) is negative? How?
