Problem 17.31.
Left–hand, infinite limits can be defined in a manner similar to Definition 17.23 and Definition 17.29.
(a)
Give a definition like Definition 17.23 for \(\llimit{x}{a}{f(x) }= \infty \)
(b)
Give a definition like Definition 17.29 for \(\llimit{x}{a}{f(x) }=-\infty \)
(c)
Use your definitions in parts (a) and (b) to prove that
\begin{equation*} \llimit{x}{a}{f(x) }=-\infty \end{equation*}
if and only if
\begin{equation*} \llimit{x}{a}{\left(-f\left(x\right)\right) }=\infty \end{equation*}
