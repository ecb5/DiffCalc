Problem 17.30.
Suppose \(f(x)\) is defined on some interval \((a,b)\text{.}\) Use Definition 17.23 and Definition 17.29 to prove that
\begin{equation*} \rlimit{x}{a}{f(x) }=-\infty \end{equation*}
if and only if
\begin{equation*} \rlimit{x}{a}{\left(-f\left(x\right)\right) }=\infty\text{.} \end{equation*}
