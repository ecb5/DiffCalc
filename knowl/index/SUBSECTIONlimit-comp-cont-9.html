Notice that in addition to changing \(\infty\) to some real number, \(a\text{,}\) we have added two qualifications to the statement of each of these theorems from Chapter 12:
  • “Suppose that the functions \(f(x)\) and \(g(x)\) are defined on some open interval about \(a\)” and,
  • “except, possibly, at \(a\) itself”
To see why these are necessary recall that we’re going to use limits to define the derivative as in Theorem 13.4 so we’ll need to evaluate the limit \(f^\prime(x)=\tlimit{h}{0}{\frac{f(x+h)-f(x)}{h}}\text{.}\) Clearly the expression \(\frac{f(x+h)-f(x)}{h}\) is not defined at \(h=0\text{.}\) But we’re only interested in its value in the limit as \(h\rightarrow0\) which means that \(h\) must be able to get close to \(0\text{.}\) That is, there must be an open interval around \(0\) where the expression \(\frac{f(x+h)-f(x)}{h}\) is defined.
