Example 4.3.5.15. Making Things “Easier on the Eyes”.

Suppose we wish to find \(\dx{y}\) when \(y=(x+2)^2(2x-3)^3\) Once again we could just apply brute force by multiplying out the expression \((x+2)^2(2x-3)^3.\) But this is a lot of work and it is very error prone.
Instead notice that \(y\) is the product of \((x+2)^2\) and \((2x-3)^3,\) so the Product Rule seems a likely place to begin. But first we make it “easier on the eyes” with the substitutions, \(z=x+2\text{,}\) and \(w=2x-3\text{.}\) Thus:
\begin{align*} y\amp =\textcolor{red}{(\underbrace{x+2}_{=z})^2}\textcolor{blue}{(\underbrace{2x-3}_{=w})^3}\\ y\amp =\textcolor{red}{z^2}\textcolor{blue}{w^3}. \end{align*}
Using the Product Rule we now have
\begin{align*} \dx y \amp = \textcolor{red}{z^2}\textcolor{blue}{\dx(w^3)} + \textcolor{blue}{w^3}\textcolor{red}{\dx(z^2)}\\ \dx y \amp = \textcolor{red}{z^2}\textcolor{blue}{\dx(w^3)} + \textcolor{blue}{w^3}\textcolor{red}{\dx(z^2)}\\ \amp = (x+2)^2\textcolor{blue}{{\dx(w^3)}} + (2x-3)^3\textcolor{red}{{\dx(z^2)}}\\ \amp = (x+2)^2\textcolor{blue}{(3w^2)\dx{w}}+(2x-3)^3\textcolor{red}{(2z)\dx{z}}\\ \amp = (x+2)^2(3)(2x-3)^2\textcolor{blue}{{\dx(2x-3)}} + (2x-3)^3(2)(x+2)\textcolor{red}{{\dx(x+2)}}\\ \amp = (x+2)^2(3)(2x-3)^2(2)\dx x + (2x-3)^3(2)(x+2)\dx{x}. \end{align*}
and so
\begin{equation*} \dx{y} = \left[6(x+2)^2(2x-3)^2 + 2(2x-3)^3(x+2)\right]\dx{x}. \end{equation*}
Not only does this involve considerably less Algebra (and is thus less prone to error) it also gives us \(\dx{y}\) in a form that is easy to factor. This is important because, as we mentioned in Comment 4.3.5.14, many of the problems to come will involve finding a differential, setting it equal to zero, and then solving the resulting equation. If we had computed this by brute force we would have to find the roots of a fully expanded fourth degree polynomial. That would be hard. The form given above is much easier to work with. Let’s go ahead and set \(\dx{y}=0\) and solve the resulting equation:
\begin{equation*} 6\textcolor{red}{(x+2)}^2\textcolor{blue}{(2x-3)}^2 + 2\textcolor{blue}{(2x-3)}^3\textcolor{red}{(x+2)} = 0. \end{equation*}
just for practice.
The prospects for solving the equation, even in this form seem pretty glum at first, but it really isn’t so bad once we notice that \(\textcolor{red}{(x+2)}\) and \(\textcolor{blue}{(2x-3)^2}\) appear as factors in both terms on the left hand side. Factoring these out gives
\begin{align} \textcolor{red}{(x+2)}\textcolor{blue}{(2x-3)^2}\left[6\textcolor{red}{(x+2)}+2\textcolor{blue}{(2x-3)}\right] \amp{} \amp{}= 0\notag\\ \end{align}
so that
\begin{align} \textcolor{red}{(x+2)}\textcolor{blue}{(2x-3)^2}\left[6x+12+4x-6)\right] \amp{}= 0\notag\\ 2(x+2)(2x-3)^2(5x+3) \amp{}= 0,\tag{4.7}\\ \end{align}
so that
\begin{align} \textcolor{red}{(x+2)}\textcolor{blue}{(2x-3)^2}\left[6x+12+4x-6)\right] \amp{}= 0\notag\\ 2(x+2)(2x-3)^2(5x+3) \amp{}= 0.\tag{4.8} \end{align}
