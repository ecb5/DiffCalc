Problem 8.4.4.

(a)

Show that
  1. \(\displaystyle \dfdx{(\cosh(x))}{x} = \sinh(x)\)
  2. \(\displaystyle \dfdx{(\sinh(x))}{x} = \cosh(x)\)
  3. \(\displaystyle \dfdx{(\tanh(x))}{x} = \sech^2(x)\)
  4. \(\displaystyle \dfdx{(\sech(x))}{x} = -\sech(x)\tanh(x)\)
  5. \(\displaystyle \dfdx{(\coth(x))}{x} = -\csch^2(x)\)
  6. \(\displaystyle \dfdx{(\csch(x))}{x} = -\csch(x)\coth(x)\)

(b)

Show that: \(\sinh(-x)=-\sinh(x)\)

(c)

Show that: \(\cosh(-x)=\cosh(x)\)

(d)

Show that: \(\sinh(x\pm y)=\sinh(x)\cosh(y)\pm \cosh(x)\sinh(y).\)

(e)

Show that: \(\cosh(x\pm y)=\cosh(x)\cosh(y) \pm \sinh(x)\sinh(y).\)

(f)

Compare each of the identities above with the corresponding trigonometric identity.
