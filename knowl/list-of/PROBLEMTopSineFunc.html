Problem 14.3.0.3.

(a)
Show that the function \(\beta(x)=\sin\left(\frac1x\right)\) does not satisfy the constraint \(\Delta\beta\neq0\) when \(x\) is near zero.
Hint.
Recall Definition 14.1.1.6.
(b)
As a result of part 14.3.0.3.a Theorem 14.3.0.2 does not apply to any of the following functions at \(x=0\text{.}\) Nevertheless one of them is differentiable at \(x=0\text{.}\) Use Definition 13.2.0.3 to find out which one.
  1. \(\displaystyle T(x)= \begin{cases} \sin\left(\frac1x\right) \amp x\neq0\\ 0 \amp x=0 \end{cases}.\)
  2. \(\displaystyle U(x)= \begin{cases} x\sin\left(\frac1x\right) \amp x\neq0\\ 0 \amp x=0 \end{cases}.\)
  3. \(\displaystyle V(x)= \begin{cases} x^2\sin\left(\frac1x\right) \amp x\neq0\\ 0 \amp x=0 \end{cases}.\)
