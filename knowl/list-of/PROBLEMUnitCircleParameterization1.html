Problem 5.9.0.8.

(a)
Show that each of the following is a parameterization of part of the unit circle.
  1. \(\displaystyle P(t)= \ParamEqThree { t} { \sqrt{1-t^2}} { -1\le t\lt 1}\)
  2. \(\displaystyle P(t)= \ParamEqThree { t} { -\sqrt{1-t^2}} { 0\le t\lt 1}\)
  3. \(\displaystyle P(t)= \ParamEqThree { \sqrt{1-t^2}} { t} { -1\le t\lt 1}\)
  4. \(\displaystyle P(t)= \ParamEqThree { \sqrt{1-t^2}} { t} { 0\le t\lt 1}\)
  5. \(\displaystyle P(t)= \ParamEqThree { t^2} { \sqrt{1-t^4}} { -1\le t\lt 1}\)
  6. \(\displaystyle P(t)= \ParamEqThree { t-2} { \sqrt{4t-(3+t^2)}} { 1\le t\lt 2}\)
(b)
Sketch only the part of the curve included in each parameterization in part (a). Be sure to indicate the direction of travel in each case.
(c)
Compute \(\frac{\dot{y}}{\dot{x}}\) and show that this yields
\begin{equation*} \dfdx{y}{x}=-\frac{x}{y} \end{equation*}
for each of the parameterizations in part (a).
