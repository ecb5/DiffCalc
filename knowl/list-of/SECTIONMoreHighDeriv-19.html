Problem 7.4.0.3.

This problem demonstrates how we can obtain a first degree polynomial that approximates \(f(x_0)\) near \(x=0\text{.}\) Suppose that \(f(x)\) is some (differentiable) function and that \(l(x) =A+Bx\text{.}\)

(a)

Show that if \(f(0)=l(0)\) then \(A=f(0).\)

(b)

Show that if \(f^\prime(0)=l^\prime(0)\) then \(B=f^\prime(0).\)
