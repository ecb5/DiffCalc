Problem 5.9.0.10.

For each of the given parameterizations find \(\dfdx{y}{x}\) two different ways:
  1. Compute \(\dx{x}\) and \(\dx{y}\) and find their ratio, \(\dfdx{y}{x}\text{.}\)
  2. Solve for \(t\) in terms of \(x\) (or \(y\)), substitute the result into \(y\) (or \(x\)), and then compute \(\dfdx{y}{x}\) directly.
And then find \(y(x)\text{.}\)
(a)
\(P(t)= \ParamEqThree {t-1} {t+1} {t\in\RR}\)
(b)
\(P(t)= \ParamEqThree { 3t-2} { -5t+7} { t\in\RR}\)
(c)
\(P(t)= \ParamEqThree { 1/\sqrt{t}} { \sqrt{t}} { 0\lt t}\)
