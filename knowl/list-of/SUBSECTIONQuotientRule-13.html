Problem 4.3.6.9.

First compute each of the following using the Quotient Rule, and then find an algebraic simplification that makes the differentiation easier. Verify that you get the same answer either way.
(a)
\(\dx{\left(\frac{x^2+x}{x}\right)}=\dx{x}\)
(b)
\(\dx{\left(\frac{x^2-1}{x+1}\right)}=\dx{x}\)
(c)
\(\dx{\left(\frac{x^3-x}{x-1}\right)}=\dx{x}\)
(d)
\(\dx{\left(\frac{x^2+x-2}{x+2}\right)}=\dx{x}\)
(e)
\(\dx{\left(\frac{x^3+2x^2-x-2}{x+2}\right)}=2x\dx{x}\)
(f)
\(\dx{\left(\frac{x^3+2x^2-x-2}{(x+2)(x-1)}\right)}=\dx{x}\)
(g)
\(\dx{\left(\frac{x^3+3x^2-x}{x^2+3 x-1}\right)}= \dx{x}\)
(h)
\(\dx{\left(\frac{x^3-x^2-2x+2}{x-1}\right)}= 2x\dx{x}\)
(i)
\(\dx{\left( \frac{x^3-x^2-x-2}{x-2}\right)}= (2x+1)\dx{x}\)
(j)
\(\dx{\left(\frac{x^3-x^2-x-2}{x^2+x+1}\right)}= \dx{x}\)
