Problem 17.2.4.9.

Suppose \(f(x)\) is defined on some interval \((a,b)\text{.}\) Use Definition 17.2.4.2 and Definition 17.2.4.8 to prove that
\begin{equation*} \rlimit{x}{a}{f(x) }=-\infty \end{equation*}
if and only if
\begin{equation*} \rlimit{x}{a}{\left(-f\left(x\right)\right) }=\infty\text{.} \end{equation*}
