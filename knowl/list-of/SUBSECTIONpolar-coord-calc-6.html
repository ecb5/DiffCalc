Problem 6.2.2.2.

Find the equation of the line tangent to the following curves at the given value of \(\theta.\)
(a)
\(r=\theta^2, \theta\ge0\)
  1. \(\displaystyle \theta=0\)
  2. \(\displaystyle \theta=\frac{\pi}{6}\)
  3. \(\displaystyle \theta=\frac{\pi}{4}\)
  4. \(\displaystyle \theta=\frac{\pi}{3}\)
  5. \(\displaystyle \theta=\frac{\pi}{2}\)
  6. \(\displaystyle \theta=\frac{37\pi}{6}\)
(b)
\(r=2+\sin(3\theta).\)
  1. \(\displaystyle \theta=0\)
  2. \(\displaystyle \theta=\frac{\pi}{6}\)
  3. \(\displaystyle \theta=-\frac{\pi}{4}\)
  4. \(\displaystyle \theta=\frac{\pi}{3}\)
  5. \(\displaystyle \theta=-\frac{\pi}{2}\)
  6. \(\displaystyle \theta=\frac{37\pi}{6}\)
(c)
\(r=\cos\left(\frac{\theta}{2}\right)\text{.}\)
  1. \(\displaystyle \theta=\frac{\pi}{6}\)
  2. \(\displaystyle \theta=-\frac{\pi}{6}\)
  3. \(\displaystyle \theta=\frac{\pi}{4}\)
  4. \(\displaystyle \theta=-\frac{\pi}{4}\)
  5. \(\displaystyle \theta=\frac{\pi}{3}\)
  6. \(\displaystyle \theta=-\frac{\pi}{3}\)
