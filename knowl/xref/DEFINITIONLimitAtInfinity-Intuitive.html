Definition 12.1. An Intuitive Definition of a Limit at \(\pm\infty\).
  1. If, as \(x\rightarrow\infty\text{,}\) \(f(x)\) gets arbitrarily close to some number \(A\text{,}\) we write
    \begin{equation*} \limit{x}{\infty}{f(x)}=A. \end{equation*}
  2. If, as \(x\rightarrow-\infty\text{,}\) \(f(x)\) gets arbitrarily close to some number \(A\text{,}\) we write
    \begin{equation*} \limit{x}{-\infty}{f(x)}=A. \end{equation*}
