Drill 8.2.

Solve each of the following IVPs. Graph your solutions:
  1. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=0\)
  2. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=1\)
  3. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=-10\)
  4. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(1)=2\)
  5. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(-1)=1\)
  6. \(\displaystyle \dfdx{y}{x}=x,\ \ \ y(0)=1\)
  7. \(\dfdx{y}{x}=2x,\ \ \ y(x_0)=y_0\) where \(x_0\) and \(y_0\) are fixed, but unspecified constants.
