Example 4.29. Brute Force Computation.
Suppose that we wish to find \(\dx{y}\) when \(y=5(x^2+x)^3\text{.}\) One way to do this is simple brute force. That is, we compute
\begin{equation*} y=5\textcolor{red}{(x^2+x)^3} = 5\textcolor{red}{(x^6+3x^5+3x^4+x^3)} = 5x^6+15x^5+15x^4+5x^3. \end{equation*}
Applying the Sum Rule we see that
\begin{equation*} \dx y = \dx \left(5x^6\right)+\dx \left(15x^5\right)+\dx \left(15x^4\right)+\dx \left(5x^3\right). \end{equation*}
Next we use the Constant Multiple Rule to get
\begin{align*} \dx y \amp{} = 5\dx \left(x^6\right)+15\dx \left(x^5\right)+15\dx \left(x^4\right)+5\dx \left(x^3\right),\\ \end{align*}
and finally the Power Rule for Positive Integers gives,
\begin{align*} \dx y \amp = 30x^5\dx x+75x^4\dx x+60x^3\dx x+15x^2\dx x. \end{align*}
In the end we have,
\begin{equation*} \dx{y}= (30x^5+75x^4+60x^3+15 x^2)\dx{x}. \end{equation*}
Brute force works but expanding an expression like \((x^2+x)^3\) is dull (and error prone) and mathematicians are like Nature: We are lazy. We’d really rather avoid doing all of that dull computation if we possibly can. Wouldn’t you?
Suppose, we make this problem a little “easier on the eyes” with a substitution like we did in Section 2.2. Let \(z=x^2+x\text{.}\) Then our original problem becomes \(y=5z^3\) which is much easier to look at. And now \(\dx{y}=15z^2\dx{z}\text{.}\) Substituting \(x^2+x\) back in for \(z\) we get \(\dx y = 15(x^2+x)^2\dx (x^2+x).\) After computing \(\dx (x^2+x) = (2x+1)\dx x\) we see that
\begin{equation*} \dx y = 15(x^2+x)^2(2x+1)\dx x. \end{equation*}
