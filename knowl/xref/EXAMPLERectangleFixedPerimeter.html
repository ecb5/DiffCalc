Example 9.87. Example 3.6 revisited again.

We first solved this problem in [cross-reference to target(s) "EXfixed-perimeter" missing or not unique] where we solved it algebraically. We revisited it the first time in Problem~\ 9.14 where we solved it again analytically (using Calculus) twice. But to do this we relied on Fermat’s Theorem and Fermat’s Theorem only allows us to find a maximum that we already know exists. At that time we did not have the tools to show that we actually had a maximum. We do now so we will solve this problem one last time before we move on.
Show that of all rectangles with a fixed perimeter, \(P\text{,}\) the one with the greatest area is a square.\\
We will follow the outline above.
  1. The objective function is
    \begin{equation*} \label{eq:FixedPerimeterObjective} A=xy, \end{equation*}
  2. The only constraint is
    \begin{equation*} \label{eq:FixedPerimeterConstraint} P=2x+2y \end{equation*}
    where \(P\) is constant.
  3. Since we are talking about a length it must be that \(x\ge0\text{.}\) From [cross-reference to target(s) "EQFixedPerimeterConstraint" missing or not unique]) we see that \(x\) must be less that or equal to \(\frac{P}{2}\text{.}\) Otherwise the perimeter will be greater than \(P\text{.}\) So the domain\aside{Strictly speaking if \(x=0\) or \(x=\frac{P}{2}\) we don’t actually have a rectangle. But it is simpler to use the Extreme Value Theorem (Theorem~\ 9.61) than to find the increasing and decreasing intervals using the First Derivative Test. Since the Extreme Value Theorem requires a closed domain we make things simpler by allowing “zero width” rectangles.} of our problem is \(0\le x\le \frac{P}{2}\text{.}\)
  4. Since the problem domain is a closed interval we will use the Extreme Value Theorem (Theorem~\ 9.61). To find the POTPs we will need \(\dfdx{A}{x}\text{.}\) From [cross-reference to target(s) "EQFixedPerimeterObjective" missing or not unique]) we see that
    \begin{equation*} \label{eq:MaxAreaDiffeq} \dx{A}=x\dx{y}+y\dx{x}. \end{equation*}
    From [cross-reference to target(s) "EQFixedPerimeterConstraint" missing or not unique]) we have
    \begin{equation*} \label{eq:FixedPerimeterDiffConstraint} \dx{y}=-\dx{x} \end{equation*}
    and also
    \begin{equation*} \label{eq:FixedPerimeterConstraintDiffeq} y=\frac12\left(P-2x\right) \end{equation*}
    Combining [cross-reference to target(s) "EQFixedPerimeterConstraint" missing or not unique], [cross-reference to target(s) "EQMaxAreaDiffeq" missing or not unique], and [cross-reference to target(s) "EQFixedPerimeterConstraintDiffeq" missing or not unique] we see that
    \begin{equation*} \dfdx{A}{x}=\frac{P}{2}-2x. \end{equation*}
    The only solution of \(\dfdx{A}{x}=0\) is \(x=\frac{P}{4}\text{,}\) so the POTPs for this problem are the end points, \(x=0\) and \(x=\frac{P}{2}\text{,}\) and \(x=\frac{P}{4}\text{.}\)
  5. The area, \(A=xy\text{,}\) will be zero when \(x=0\) and when \(x=\frac{P}{2}\) (why?) so by the Extreme Value Theorem the maximum occurs when \(x=\frac{P}{4}\text{.}\)
  6. This problem asks us to show that the maximal rectangle must be a square, so we need to show that \(x=y\text{.}\) We know that the maximum occurs when \(x=\frac{P}{4}\text{.}\) Plugging this value into [cross-reference to target(s) "EQFixedPerimeterConstraintDiffeq" missing or not unique] we see that \(y\) is also equal to \(\frac{P}{4}\text{.}\) Thus the maximal rectangle is a square as stated.
