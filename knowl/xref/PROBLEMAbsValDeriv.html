Problem 16.5.

(a)
The Absolute Value function is not differentiable at \(x=0\) but it is differentiable at every other value of \(x\text{.}\) Let \(f(x)=\abs{x}\) and use Definition 13.4 to show that:
  1. If \(x\gt0\) then \(f^\prime(x)=1\text{,}\)
  2. If \(x\lt0\) then \(f^\prime(x)=-1\text{.}\)
(b)
From the discussion in DIGRESSION: The Absolute Value Function we know that \(f(x)=\abs{x}=\sqrt{x^2}\text{.}\) Use the differentiation rules you learned in Part 1 of this text to show that if \(x\neq0\text{,}\) then \(f^\prime(x) = \frac{x}{\abs{x}}.\) How does this compare with your calculations part 16.5.a?
