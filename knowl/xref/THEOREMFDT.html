Theorem 9.41. The First Derivative Test.
Comment.
This is a rather naive statement of this theorem, but it will suffice for our current purposes. We will revisit, restate, and prove the First Derivative Test in SectionÂ 15.3
Suppose \(f\) is a function of \(x\text{.}\) The following statements are true:
  1. If \(f^\prime(a)\gt 0\text{,}\) on some interval then \(f(x)\) is increasing on the interval.
  2. If \(f^\prime(a)\lt 0\text{,}\) on some interval then \(f(x)\) is decreasing on the interval.
